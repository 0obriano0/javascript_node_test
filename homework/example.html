簡單

靜態

只用 Markdown (或 Textile)、Liquid、HTML & CSS 就可以構建可部署的靜態網站。

Jekyll 模版指南 →

部落格形態

自定義地址、分類、頁面、部落格內容 以及 自定義的佈局設計 都是系統中的一等公民。

遷移你的部落格 →

只需幾秒鐘就能讓網站跑起來。

就這幾個指令

~ $ gem install jekyll

~ $ jekyll new my-awesome-site

~ $ cd my-awesome-site

~/my-awesome-site $ jekyll serve

# → Now browse to http://localhost:4000
